# Signal Rat: Cyberdeck RPG Manual

Genre: flagship fictional cyberdeck RPG.

## Overview

Signal Rat is a focused cyberdeck fantasy about safehouse prep, faction jobs, simulated nodes, recovered files, deck upgrades, heat, trace, and final signal-chain pressure.

## Goal Of A Run

Complete faction jobs, recover files, upgrade the deck, and unlock final-chain conditions before heat, battery loss, deck damage, or debt pressure ends the run.

## Controls

Use number keys to choose visible actions. Move with `W/S`, `K/J`, or `;/.`. Page with `A/D`. Select with `Enter` or BtnA. Back out with `Q` or Backspace.

## Saving, Loading, And Launcher Return

## Screen Guide

### SAFEHOUSE

SAFEHOUSE handles Streetline, Neon Crown, Ghost Relay, debt, blackbox, patch, fence, and clinic leads.

### JOB BOARD

JOB BOARD offers jobs such as Rat Trap, Ledger Node, Crown Cache, Relay 17, Clinic Queue, Dead Mall, Servo Pawn, and Weather Feed.

### NODE

NODE runs fictional commands: scan, decrypt, spoof, inject, trace, disconnect, blackbox, and scrub. Every command changes simulated node pressure only.

### FILES

FILES recovers ghost echo, crown debt, street map, relay pact, clinic bill, weather lie, mall tape, servo key, old signal, and final chain story pieces.

### DECK

DECK upgrades cold CPU, spoof chip, battery stack, and blackbox drive modules.

## Core Loop

1. Rest and choose leads at SAFEHOUSE.
2. Take a faction job.
3. Run NODE commands until success or disconnect.
4. Recover FILES when possible.
5. Upgrade DECK modules.
6. Manage heat and final-chain flags.

## Risk, Progress, And Endings

Heat and battery limit how far to push a job. Deck damage and ghost pressure can create failure paths. Endings include clean ghost, rich crown, street legend, relay pact, debt lock, and deck burnout.

## Tips For First Run

- Upgrade battery stack or cold CPU early.
- Use disconnect before a job spirals.
- Recover files when heat is manageable.
- Neon Crown style choices may pay more but increase risk.
- Blackbox choices matter for final-chain progress.

## Offline Fictional Scope

Every contract, node, trace, ghost signal, file, scan, decrypt, spoof, inject, blackbox, and deck upgrade is compiled fiction. Signal Rat never touches real devices, networks, radios, credentials, packets, Wi-Fi, Bluetooth, sockets, or the internet.
